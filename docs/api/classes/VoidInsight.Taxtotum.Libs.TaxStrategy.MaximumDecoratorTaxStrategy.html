MaximumDecoratorTaxStrategy

Extends \VoidInsight\Taxtotum\Libs\TaxStrategy\AbstractDecoratorTaxStrategy

Strategy to limit maximum computated tax to a defined value.

Decorator pattern is used to adjust computation algorithm of Strategy and reduce numbers of Strategy needed.
